Introducción al Frontend y Backend

En esta investigación/proyecto inicié mi aprendizaje en el área del Frontend, que corresponde a la parte de una aplicación web que los usuarios pueden ver y con la que interactúan, utilizando tecnologías clave como HTML, CSS y JavaScript. Estas herramientas son fundamentales para ofrecer una experiencia de usuario atractiva y eficaz, ya que cada una aporta a la estructura, el diseño y la interactividad del sitio web. Después, decidí centrarme en el Backend, el cual opera en el servidor y se ocupa de la gestión de bases de datos, la autenticación de usuarios y el procesamiento de datos a través de lenguajes como PHP, Python y Java. Este estudio me permitió entender cómo el Backend se encarga de que las acciones del usuario se ejecuten correctamente y que el servidor responda de manera eficiente. Asimismo, investigué la distinción de roles entre el frontend y el backend, lo que favorece un control más eficaz del flujo de información y garantiza que los usuarios no tengan acceso directo a la lógica interna o a las bases de datos. También analicé los lenguajes que se utilizan en cada área, destacando que el frontend se basa en HTML, CSS y JavaScript, mientras que el backend utiliza lenguajes como PHP, Python y Java, entre otros. Por último, estudié diferentes stacks de desarrollo, como LAMP y MEAN, que combinan varias tecnologías para facilitar el desarrollo de aplicaciones web. Al concluir esta investigación, entendí la importancia del desarrollo Full Stack, que incluye tanto el Frontend como el Backend. Este enfoque me brindó una nueva perspectiva sobre las oportunidades en el desarrollo de proyectos web, donde la integración y el conocimiento de ambas áreas son cruciales para crear experiencias completas y funcionales para los usuarios.

HTML5

Este documento ofrece un análisis profundo de HTML5, la última versión del lenguaje de marcado estándar para sitios web. Se destacan las innovaciones que trae, como la integración nativa de contenido multimedia, que elimina la necesidad de complementos externos, y nuevos elementos semánticos como "header" , "footer" , "article" y "section" , que facilitan una estructura más lógica. y accesible. El objetivo principal es mostrar cómo estas características mejoran la experiencia del usuario y permiten crear aplicaciones web más sólidas y adaptativas. También se aborda la importancia de la accesibilidad, subrayando cómo HTML5 contribuye a hacer los sitios más inclusivos para personas con discapacidades. El documento incluye ejemplos prácticos sobre el uso de etiquetas y atributos de HTML5, así como una guía sobre la estructura de un documento, destacando la relevancia de los metadatos para el SEO y la usabilidad. La relevancia de este estudio radica en la necesidad de recursos que aborden tanto lo fundamental como las demandas actuales de la industria. Al comparar diferentes fuentes sobre HTML5, se identifican sus fortalezas y debilidades, proporcionando criterios útiles para los desarrolladores en formación. En resumen, este análisis no solo se centra en las características técnicas de HTML5, sino que también considera su impacto en la experiencia del usuario y la accesibilidad. Se espera que los lectores adquieran una comprensión profunda sobre cómo utilizar HTML5 de manera efectiva, convirtiendo este documento en un recurso valioso para quienes dominan el desarrollo web moderno.

EDPT

El propósito de este proyecto es concebir y desarrollar la aplicación "EDPT" (EduPlayTime), dirigida a niños de 1 a 5 años, con un enfoque en el aprendizaje interactivo y supervisado por los padres. "EDPT" busca no solo proporcionar actividades educativas, sino también ofrecer a los padres un control efectivo sobre el tiempo de juego de sus hijos a través de un sistema de gestión de tiempo personalizable. La importancia de abordar esta problemática se fundamenta en estudios previos que destacan los posibles impactos negativos del uso excesivo de dispositivos móviles en el desarrollo cognitivo y emocional de los niños. Al ofrecer un catálogo diverso de actividades educativas y juegos interactivos. Este proyecto aspira a contribuir significativamente al ámbito de la educación digital temprana, proporcionando a los niños una experiencia educativa enriquecedora y a los padres una herramienta confiable para guiar el uso de la tecnología. La aplicación "EDPT" se perfila como una oportunidad para sentar las bases de un desarrollo saludable en la era digital.
